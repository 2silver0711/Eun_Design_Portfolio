# Portfolio Website

비트/코드 느낌의 독보적인 포트폴리오 사이트

## 🎨 디자인 특징

- **비트/코드 스타일**: ASCII 아트, 도트 매트릭스, 할프톤 효과
- **글리치 인터랙션**: 네비게이션과 인터랙티브 요소에 글리치 효과
- **Space Mono 폰트**: 모노스페이스 타이포그래피
- **네온 컬러**: 프로젝트별 포인트 컬러
  - STOOV: `#FF4800`
  - LocalLink: `#00DC6E`
  - Budget Buddy: `#00D4FF`

## 📱 반응형 디자인

- 데스크톱: 1960px 기본 너비
- 태블릿: 768px ~ 1199px
- 모바일: ~ 767px

## 🚀 배포

이 프로젝트는 GitHub Pages로 자동 배포됩니다.

자세한 배포 방법은 [DEPLOYMENT.md](./DEPLOYMENT.md)를 참고하세요.

## 🛠️ 기술 스택

- **Framework**: React 18.3.1
- **Build Tool**: Vite 6.3.5
- **Styling**: Tailwind CSS 4.1.12
- **Routing**: React Router 7.13.0
- **Language**: TypeScript
- **UI Components**: Radix UI
- **Animations**: Motion (Framer Motion)

## 💻 로컬 개발

### 요구사항

- Node.js 20+
- pnpm 8+

### 설치 및 실행

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm run dev

# 프로덕션 빌드
pnpm run build
```
